# APP Precio Luz

Aplicación móvil para consultar el precio de la electricidad en España en tiempo real.

## Estructura del proyecto

```
APP Precio Luz/
├── LuzApp/           # Aplicación principal (React Native + Expo)
│   ├── src/          # Código fuente
│   ├── android/      # Archivos nativos Android
│   └── backend/      # Backend Node.js
├── Documento de Especificación de Requerimientos.md
└── diseño_ux.png     # Diseño UX
```

## Quick Start

```bash
cd LuzApp
npm install
cd backend && npm install && cd ..
npm run dev
```

Para más información, consulta el README.md dentro de la carpeta LuzApp.
